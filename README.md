# Code Retreat Skeleton Code

This repository contains skeleton code that Code Retreat Participants can use to get started.

## Go

```shell
# Update BUILD files
./pleasew puku fmt //go/...

# Run the binary
./pleasew run //go/cmd/conway

# Run the tests
./pleasew test //go/...
```

## Python

```shell
# Run the binary
./pleasew run //python:main

# Run the tests
./pleasew test //python/...
